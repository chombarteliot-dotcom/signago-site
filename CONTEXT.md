## L'offre — Catalogue (6 catégories)
1. **Protection & Barriérage** 🛡️ — clôtures grillagées (Heras), clôtures bardées, barrières Vauban, barrières standard, portails, séparateurs béton GBA
2. **Franchissement** 🌉 — passerelles piétons, passages câbles, plaques de roulage légères/lourdes, ponts de tranchée
3. **Signalisation temporaire** 🚧 — panneaux AK/BK/KC/KD, directionnels, supports & mâts
4. **Signalisation lumineuse** 💡 — feux tricolores, feux d'alternat, balises & gyrophares, flash
5. **Balisage & Séparation** 🔶 — cônes & balises K5, séparateurs Pyraneo/K16, rubans, balisage événementiel
6. **Accessoires & Lestage** ⚙️ — plots béton, stabilisateurs, fixations, bâches, stockage

## Objectif du site
Site vitrine BtoB — rassurer les prospects, présenter le catalogue, générer des demandes de devis.
**Aucune vente en ligne.** Chaque fiche produit → bouton "Demander un devis".

## Règles éditoriales (non négociables)
- Aucun placeholder / Lorem ipsum / témoignage inventé en ligne
- Chaque fiche produit : photo HD + dimensions + contexte d'utilisation + conformité réglementaire + bouton devis visible sans scroller
- Promesse chiffrée affichée partout : "Devis sous 4h — Livraison sous 24h sur Dunkerque"
- Navigation catalogue 100% mobile (clients sur chantier, téléphone, mains sales)
- Section ressources réglementaires (guides, normes NF, arrêtés de voirie, checklists) — SEO différenciateur
- SEO local : pages dédiées Dunkerque, Calais, Boulogne-sur-Mer, Saint-Omer
- Site bilingue FR/EN envisagé (acteurs étrangers présents sur Dunkerque)

## Stack technique
- HTML / CSS / JS pur (pas de framework)
- GitHub + GitHub Pages ou Netlify (hébergement gratuit)
- Formspree (formulaire de devis)
- Domaine : OVH
